# Hypercycles, 1995

This repository is a mirror of a port of **Hypercycles**, a DOS game developed by *Aclypse software* and released in 1995.

Note that this repository is a *WIP*

## Misc
### Build
```console
./make.sh
```

### Clean
```console
./make.sh clean
```
